# BIZ AI Assistant

## 기업용 AI 어시스턴트 솔루션

Windows PC 한 대로 구축하는 기업 맞춤형 AI 비서.
데이터 분석, 문서 자동화, 업무 보고를 채팅 한 줄로 처리.

---

## 핵심 콘셉트

> "기업 현장에 PC 한 대 깔아주면, AI가 데이터를 분석하고 보고서를 쓴다"

- 직원이 파일을 공유폴더에 넣거나 텔레그램으로 보내면
- AI가 자동으로 분석/분류/정리
- 결과를 텔레그램 or 보고서 파일로 회신
- 정기 리포트도 크론잡으로 자동 생성

---

## 제공 기능

### 1. 데이터 분석
- Excel/CSV/JSON 파일 분석
- 이상치 탐지, 패턴 분석, 트렌드 파악
- 매출/재고/인사 데이터 등 업종별 맞춤 분석
- 시각화 차트 자동 생성

### 2. 문서 자동화
- 보고서 자동 작성 (일일/주간/월간)
- 회의록 정리
- 계약서/견적서 초안 생성
- 한글(HWP)/엑셀/PDF 변환

### 3. 파일 분류 & 정리
- 공유폴더 파일 자동 분류
- 문서 내용 기반 태깅
- 중복 파일 탐지
- 폴더 구조 자동 정리

### 4. DB 연동 분석
- 기업 DB(MySQL/PostgreSQL/MSSQL) 직접 연결
- 자연어로 쿼리 ("이번 달 매출 상위 10개 거래처 뽑아줘")
- 정기 모니터링 & 이상 알림

### 5. 커뮤니케이션
- 텔레그램 봇으로 실시간 요청/응답
- 정기 보고서 자동 발송
- 팀별 채널 분리 가능

---

## 시스템 구조

```
Windows PC (기업 현장)
│
├── WSL2 (Ubuntu)
│   ├── Claude Code (AI 엔진)
│   ├── 텔레그램 봇 (채팅 인터페이스)
│   ├── 크론잡 (자동 스케줄)
│   │   ├── 매일 09:00 — 일일 리포트 생성
│   │   ├── 매주 월 — 주간 분석 보고
│   │   └── 파일 감시 — 새 파일 자동 처리
│   └── 분석 도구 (Python, Node.js)
│
├── 공유 폴더 (SMB)
│   ├── /입고/ — 직원이 파일 넣는 곳
│   ├── /결과/ — AI가 결과 저장하는 곳
│   └── /보고서/ — 자동 생성 보고서
│
├── DB 연결 (선택)
│   └── 기업 ERP/회계 시스템 연동
│
└── 원격 관리 (RDP)
    └── 외부에서 세팅/유지보수
```

## 업종별 활용 예시

### 건설업
- 일일 작업일보 자동 정리
- 자재 입출고 데이터 분석
- 안전서류(TBM 등) 자동 생성
- 공정률 자동 계산 & 보고

### 제조업
- 생산 데이터 이상치 탐지
- 불량률 추이 분석
- 재고 현황 리포트
- 거래처별 매출 분석

### 도매/유통
- 매출/매입 자동 대조
- 거래처별 미수금 알림
- 상품별 판매 추이
- 재고 부족 예측

### 사무/서비스
- 이메일/문서 분류
- 회의록 자동 작성
- 고객 문의 분류 & 통계
- 직원 근태 데이터 정리

## 기술 스택

- **OS**: Windows 11 + WSL2 (Ubuntu)
- **AI**: Claude Code (Anthropic)
- **커뮤니케이션**: Telegram Bot API
- **데이터 분석**: Python (pandas, matplotlib)
- **자동화**: Cron, inotifywait (파일 감시)
- **DB**: 기업 기존 DB 연동 (MySQL/PostgreSQL/MSSQL/SQLite)
- **문서**: python-docx, openpyxl, pyhwp
- **원격 관리**: SSH, RDP

---

## 세팅 요구사항

### 하드웨어
- Windows 10/11 PC (최소 사양)
- RAM: 8GB 이상 (16GB 권장)
- 저장공간: SSD 256GB 이상
- 인터넷: 유선 연결 권장

### 소프트웨어
- WSL2 + Ubuntu
- Claude Code CLI
- Python 3.10+
- Node.js 18+

### 비용
- PC: 기업 기존 PC 활용 가능
- Claude API: 사용량 기반 (월 $20~100 예상)
- 기타: 무료 오픈소스

## 도입 프로세스

```
1단계: 현장 분석 (1일)
   └── 업무 파악, 데이터 흐름 확인, 니즈 정리

2단계: 세팅 (1~2일)
   └── PC 설정, WSL2/Claude 설치, 텔레그램 봇 연결

3단계: 맞춤 설정 (2~3일)
   └── 업종별 분석 템플릿, 보고서 양식, 크론잡 설정

4단계: 교육 (1일)
   └── 직원 대상 사용법 교육, 매뉴얼 제공

5단계: 운영 & 유지보수
   └── 원격 모니터링, 월간 최적화
```

## 보안 고려사항

- 데이터는 분석 시에만 AI 서버로 전송, 저장하지 않음
- Anthropic 기업용 정책: 입력 데이터 학습에 사용하지 않음
- 민감 데이터는 로컬 처리 옵션 제공
- VPN/방화벽 내 운영 가능
- 접근 권한 관리 (텔레그램 allowlist)
